SELECTED PROJECTS


Robust Navigation System for Quadruped Robots || C++, EKF, Perception, Controls, Gazebo

  • Constructed a 2 stage odometry data pipeline that takes in RGBD odometry followed by refinement using point to plane ICP algorithm. The localization technique reduced the error to 2% and brought down the computation time to 0.1s.
  • Researched a combination of state estimation and point cloud odometry to develop a navigation system.
  • Formulated a state estimator using a novel slope estimation technique, IMU and motor feedback data, the state was fused with the point cloud odometry data. Developed an optimized Open3D – ROS bridge for faster computation

CNN Depth estimation || Python, Deep Learning, Perception

  • Developed a new depth estimation technique by combining Sum of absolute difference, and Census transform by using Laplacian equation to minimize the error to 2.5% from 7%.
  • Enhanced the depth image by using RGB and gradient imagesfor more spatial information, this combined with Multi block matching reduced the error 5%.
  • Modeled a new convolutional neural network called Y-net, which brought down the error to 1%. The model used a state-of-the-art convolutional block derived from multiblock matching.

Gait Transition for Quadrupeds || MPC, C++, Mujoco

  • Created an hierarchical MPC for gait transition, we used discretized foot step and the initial state and final state and then plan into the future. Using the fact that we have multiple final states we fit multiple MPC, we finally pick the one with least cost.
  • The project was written in C++ and we used mujoco as a simulation tool to visualize our results.

Control of Autonomous Janitorial Robot || C++, Python, Kalman Filter, ROS

  • Built and simulated a robot to clean a washroom for the ARTPARK robotics challenge and secured funding of $5300.
  • Wrote the algorithm to enter arena by utilizing wall following and optimal entry point using door detection and 2D projection of wall onto the floor.
  • Determined the positions of coke cans, markings and dustbins using YoloV4 with 96% accuracy, followed by identifying the position of tabletop and sink using plane segmentation.
  • Calculated the best order of picking the objects and cleaning the washroom that reduced the time of completion by 5mins.
